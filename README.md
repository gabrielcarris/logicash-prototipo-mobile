# LogiCash - Prototipo mobile

Prototipo navegavel em HTML, CSS e JavaScript puro para a atividade de prototipacao da FIAP.

O LogiCash e uma plataforma SaaS B2B para transportadoras de pequeno e medio porte que operam last-mile delivery para marketplaces e e-commerces. O prototipo mostra o fluxo financeiro da operacao: importacao de pre-fatura, conciliacao de repasses, analise de divergencias, pagamento de motoristas, margem por rota/cidade/marketplace e relatorios.

## Arquivos

- `index.html`: prototipo navegavel mobile-first, sem backend e sem dependencias externas.
- `telas/`: prints verticais das telas do prototipo para uso no documento da atividade.
- `docs/atividade_prototipacao_logicash.docx`: documento da atividade preenchido.
- `docs/logicash_telas_prototipo.zip`: pacote com os prints das telas.

## Telas

1. Login mobile
2. Inicio / dashboard financeiro
3. Importar pre-fatura
4. Conciliacao de repasses
5. Detalhes da divergencia
6. Motoristas / pagamento de motoristas
7. Margem
8. Relatorios
9. Mais
